# HACKER-TOOLS-

# FSOCIETY PROTOCOL

Central de ferramentas para estudos de segurança, monitoramento de rede e automação no Termux.

## Funções
- Scanner de rede
- Monitoramento Wi-Fi
- Geolocalização de IP
- Verificação de malware
- Ferramentas de análise
- Integração com Tor

## Instalação

pkg update && pkg upgrade -y

pkg install python git curl nmap tor tcpdump clamav -y

pip install -r requirements.txt

## Executar

python main.py
